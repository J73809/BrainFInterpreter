# Brainf* Interpreter in C#

A simple Brainf* interpreter written in C#.  
Supports loops, pointer movement, cell increment/decrement, and input/output.

---

## Features

- Fully functional BF interpreter
- 30,000-byte memory tape
- Handles nested loops with a jump table
- Console-based input and output
- Lightweight and easy to run

---

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/J73809/BrainFInterpreter.git
````

2. Open the project
3. Run the program:

```bash
dotnet run
```

4. Enter your Brainf* code when prompted
